# 方案对比评分标准

AtlasAgent Phase 4 多方案对比表使用本 rubric。

## 星级维度（1–5 ⭐）

| 维度 | 5⭐ | 3⭐ | 1⭐ |
|------|-----|-----|-----|
| **浪漫指数** | 私密体验、夜景、慢节奏 | 混合 | 纯打卡赶路 |
| **美食指数** | 米其林/本地名店/美食街 | 有亮点 | 便利店解决 |
| **出片指数** | 经典机位+小众角度 | 有几个点 | 不适合拍照 |
| **文化深度** | 博物馆/历史/本地体验 | 部分文化 | 纯购物 |
| **亲子友好** | 推车友好、互动项目 | 部分适合 | 体力/内容不适合 |

## 非星级维度

| 维度 | 选项 |
|------|------|
| 体力消耗 | 低 / 中 / 高 |
| 签证复杂度 | 低（免签）/ 中（电子签）/ 高（使馆面签） |
| 飞行/铁路时间 | 具体小时数 |
| 人均预算 | 用户偏好币种 |

## 推荐排序规则

1. 若用户明确偏好（美食/浪漫/亲子），该维度权重 ×2
2. 预算超限方案标注 ⚠️ 并排后
3. 签证来不及的方案标注 ❌ 除非用户已有签
4. 最终推荐 1 个首选 + 1 个备选，附 1–2 句理由
