# 目的地证件附加要求

## 日本
- Visit Japan Web（入境审查，2024+）
- 免税购物需护照

## 美国
- ESTA 确认打印（如适用）
- REAL ID 国内段（美国公民）

## 欧盟申根
- 旅行保险 ≥ €30,000
- 携带护照/签证原件（边检）

## 澳大利亚
- 入境申报卡（动植物/食品）
- SmartGate 适用国

## 中东（UAE/沙特等）
- 部分国家着装要求
- 酒精/药物入境限制

## 非洲（ safari 目的地）
- 黄热病证书（来自疫区或经停）
- 疟疾预防药咨询

## 潜水/登山
- PADI 证书
- 登山许可（如富士山、乞力马扎罗）
