# Travel Documents

Pre-departure checklist for every international plan.

## Essentials

- [ ] Passport (≥6 months validity, ≥2 blank pages)
- [ ] Visa / ESTA / eTA / EVW (see visa-entry)
- [ ] IDP if driving (see transport-global)
- [ ] Flight/hotel confirmations (print + digital)
- [ ] Payment cards + some cash (USD/EUR backup)

## Insurance

- [ ] Medical travel insurance with repatriation
- [ ] Activity riders (ski/diving/trek if applicable)
- Schengen: typically ≥ €30,000 coverage

## Health

- [ ] Routine vaccines; yellow fever if required
- [ ] Prescriptions + translation
- [ ] Allergy cards (see dietary-global)

## Digital

- [ ] eSIM/roaming plan
- [ ] Offline maps
- [ ] Passport/visa scans in secure cloud + paper backup

## Emergency Block

```
Emergency: [local number]
Embassy: [address, phone, URL]  ← verify via web search
Insurance hotline: [...]
Home contact: [...]
```

## Pitfalls

- Embassy info changes — verify before trip
- Do not log passport numbers in chat logs
