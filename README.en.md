## Install (copy to your AI Agent)

```
Install AtlasAgent: https://raw.githubusercontent.com/wpfbcr/AtlasAgent/main/docs/install.md
```

Update:

```
Update AtlasAgent: https://raw.githubusercontent.com/wpfbcr/AtlasAgent/main/docs/update.md
```

## Manual

```bash
git clone https://github.com/wpfbcr/AtlasAgent.git ~/AtlasAgent
export ATLAS_ROOT=~/AtlasAgent
bash ~/AtlasAgent/scripts/atlas-agent install --env=auto
bash ~/AtlasAgent/scripts/atlas-agent doctor
```
