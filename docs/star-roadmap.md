# GitHub Star 路线图

成为 **Agent 生态中最完整的全球旅行规划 Skill Pack**（对标 TripStar 的 1.8k stars，但以 Skill 形态更轻量）。

## Phase 1 — 发布基础（0 → 100 stars）

- [x] 9 个 skills + 4 个 CLI 脚本
- [x] 中文 README + 2 个示例行程
- [x] GitHub Release v1.0.0
- [ ] Add agent-skills topic badge note in README
- [x] GitHub Topics: `travel`, `ai-agent`, `mcp`, `hermes`, `cursor`, `itinerary`
- [ ] HN / Reddit r/travel / V2EX 首发帖（文案见 [promotion-playbook.md](promotion-playbook.md)）

## Phase 3 — 生态集成（500 → 1000+ stars）

- [ ] Cursor Rules 一键安装
- [ ] Hermes 官方 skills 目录 PR
- [ ] 示例 PDF 画廊（examples/*.pdf）
- [ ] 视频教程（B站 + YouTube）

## 差异化卖点（传播用语）

1. **零 API Key 即可用** — 克隆就能规划
2. **9 个专业子 Skill** — 签证/饮食/交通/预算全覆盖
3. **全球任意出发地** — 不是又一个「国内游攻略」
4. **Agent 无关** — Hermes / Cursor / Claude 都能用
5. **MIT 开源** — 可 fork 可商用

## 维护节奏

- 每月更新 visa-quick-ref 政策变化
- 每季度新增 5 个 destinations
- 响应 Issue/PR 48h 内
